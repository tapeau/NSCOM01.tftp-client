# EASY TFTP
A simple command-line TFTP client application developed as Machine Project #1 for DLSU NSCOM01 course (T2 2023-2024)

## Members
- John Lorenzo Tapia
- Raico Luis Madrinan

## Running the client
1. Install required packages
```
pip install -r requirements.txt
```

2. Build and run the client
```
python EASY_TFTP.py
```
